Electronは、HTML、CSS、JavaScriptを使ったデスクトップアプリを作ることができるフレームワークです。

Windows環境で、Electron + React を利用するための環境セットアップ方法を残しておきます。

## 1. 前提

- Node.js をインストール済み

## 2. npm にアクセスするためのProxy設定

```powershell
npm config set strict-ssl false
npm config registry "http://registry.npmjs.org"
npm config set proxy "利用しているProxyのURL"
npm config set https-proxy "利用しているProxyのURL"
```

## 3. Electron をインストールするための環境変数設定

```powershell
$env:ELECTRON_GET_USE_PROXY = 1
$env:GLOBAL_AGENT_HTTPS_PROXY = "利用しているProxyのURL"
```

## 4. Electron をインストール

GitHub の electron-react-boilerplate をクローンします。

```powershell
git clone --depth 1 --branch main https://github.com/electron-react-boilerplate/electron-react-boilerplate your-project-name
cd your-project-name
npm install 
```

> `your-project-name` は、適切な名前に変更してください。
