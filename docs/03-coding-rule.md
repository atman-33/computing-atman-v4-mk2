## Nextjs

### フォルダ名とファイル名

ファイル名とフォルダー名の単語を区切るのに小文字とハイフンを使用する（ケバブケース）。

ex: 
- フォルダ: about, blog
- ファイル: sitemap.ts

### コンポーネント、コンテキスト

名前はPascalCaseを使用する。

ex: 
- Button.tsx
- ThemeContext.tsx
